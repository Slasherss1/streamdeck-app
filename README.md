# Instalation
## 1. From source code
### You will need node.js for this
1. Download the project from GitHub
2. Open cmd or powershell in folder with project
3. Type in: `npm install`
4. To start project enter `npm t`
## 2. From release
1. Download
2. Un-zip
3. Launch *.exe
